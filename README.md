# gemini-canvas-mission

Gemini Canvas 미션 결과물을 관리하는 저장소입니다.

## 목적

- Gemini Canvas로 제작한 웹앱 결과물을 기록하고 공유합니다.
- 앱별 요구사항, 배포 링크, 회고를 일관된 형식으로 남깁니다.

---

## 앱 이름: 나의 냉장고를 부탁해
## 카테고리: 유틸리티 앱

### 배포 링크
https://gemini.google.com/share/4f1f4f4daba0

### 이 앱을 만든 이유
* 냉장고에 재료는 남아있지만 막상 무엇을 요리해야 할지 떠오르지 않아 결국 배달 음식을 시키거나 재료의 유통기한이 지나 버리게 되는 문제를 해결하고자 했습니다.
* 장을 보기 전 냉장고를 비워야 할 때 또는 가지고 있는 제한적인 재료로 색다른 요리를 시도해보고 싶을 때 언제 어디서든 간편하게 사용합니다.

### 주요 기능
* **재료 입력 기능**: 사용자가 가진 재료를 콤마(,)로 구분해 한꺼번에 입력하고 시각적인 태그 형태로 관리할 수 있습니다.
* **다국적 레시피 제안**: 한식에 국한되지 않고 일식, 중식, 양식, 동남아식 등 8가지의 다양한 요리 카테고리를 추천합니다.
* **상세 조리 가이드**: 난이도, 예상 소요 시간, 추가로 필요한 최소한의 재료, 그리고 단계별 조리법을 상세히 제공합니다.
* **반응형 모달 상세 보기**: 선택한 레시피의 상세 정보를 깔끔한 레이아웃의 팝업창으로 확인할 수 있어 보기 편합니다.

### AI 기능
* **Gemini AI 레시피 생성**: 단순히 데이터베이스에서 검색하는 것이 아니라, Gemini AI 2.5 Flash 모델을 연동하여 사용자가 입력한 재료 조합을 분석하여 사용자의 재료에 맞춰 실시간으로 가능한 최적의 요리를 추천합니다.

## 앱 이름: 5초 챌린지
## 카테고리: 게임

### 배포 링크
https://gemini.google.com/share/8a4d348d8dae

### 이 앱을 만든 이유
* 누구나 쉽고 재미있게 즐길 수 있는 미니 게임을 제공하여, 자신의 시간 감각이 얼마나 정확한지 확인해보고자 했습니다.
* 친구나 가족과 가벼운 내기를 할 때, 혹은 집중력을 높이기 위해 자신의 감각을 테스트하고 싶을 때 언제 어디서나 간편하게 할 수 있습니다.

### 주요 기능
* **시간 숨김 타이머**: 시작 버튼을 누르면 시간이 흐르지만, 화면에는 "???"로 표시되어 오직 마음속으로만 시간을 세도록 설계되었습니다.
* **정밀한 결과 측정**: 정지 버튼을 누르는 즉시 실제 흐른 시간과 목표 시간(5초) 사이의 오차를 소수점 둘째 자리까지 정밀하게 계산합니다.
* **점수 및 등급 시스템**: 오차 범위에 따라 최대 100점까지 점수를 부여하며, 결과에 따라 맞춤형 코멘트를 제공합니다.
* **재도전**: '다시 도전' 버튼을 통해 초기 상태로 즉시 돌아가 반복해서 게임을 즐길 수 있습니다.

## 앱 이름: AlgoStrategy
## 카테고리: 페어 프롬프트 릴레이

### 배포 링크
https://gemini.google.com/share/2fc13e624834

### 이 앱을 만든 이유
- 코딩 테스트 준비 시 모든 문제를 직접 구현하는 방식은 물리적 시간 소모가 커서 다양한 유형을 섭렵하기 어렵다는 한계가 있습니다. 코드를 구현하는 것도 중요하지만 문제의 제약 조건을 분석해 최적의 자료구조와 알고리즘을 설계하는 안목이 가장 중요하다고 생각했습니다.
- **활용 방안**: 이 앱은 구현 과정을 생략하고 **'문제 해석 → 제약 조건 분석 → 최적 자료구조 및 알고리즘 선택'**이라는 핵심 전략 수립 단계에만 집중하게 함으로써, 짧은 시간 동안 폭넓은 유형의 문제를 압축적으로 학습할 수 있도록 돕습니다.

### 주요 기능
* **난이도별 맞춤 문제 탐색**: 초급, 중급, 고급으로 사용자가 원하는 수준에 맞는 문제를 실시간으로 불러옵니다.
* **실전 사이트 연동**: 백준(BOJ), 프로그래머스 등 실제 코딩 테스트 플랫폼의 실존 문제와 원문 링크를 제공합니다.
* **알고리즘 전략 퀴즈**: 문제의 제약 사항(N의 범위 등)을 보고 여러가지 후보 중 최적의 알고리즘 기법을 선택하는 인터랙티브 퀴즈를 제공합니다.

### AI 기능
* **실시간 문제 큐레이션 (Gemini 2.5 Flash)**: 구글 검색 기능을 활용하여 데이터베이스에 고정된 문제가 아닌, 웹상의 최신 기출 문제를 실시간으로 검색하고 난이도 기준에 맞춰 추출합니다.
* **심층 전략 분석 리포트**: 사용자가 선택한 답변의 정답 여부뿐만 아니라, **시간 복잡도 계산 근거, 단계별 알고리즘 설계 프로세스, 핵심 마스터리 개념**을 AI가 즉석에서 생성하여 심도 있는 피드백을 제공합니다.
* **중복 방지 로직**: 사용자가 이미 풀었던 문제 제목을 기억하여 새로운 문제를 우선적으로 제시하도록 지능적으로 필터링합니다.

## 앱 이름: Java Stream Quiz
## 카테고리: 학습 앱

### 배포 링크
https://gemini.google.com/share/f8df571ec196

### 이 앱을 만든 이유
자바 스트림(Stream) API는 현대 자바 개발의 핵심이지만, 모든 코드를 직접 타이핑하며 연습하기에는 시간이 너무 많이 소요된다는 불편함이 있습니다. 해당 앱을 사용하면 빈칸 채우기 형식을 통해 짧은 시간 안에 수많은 스트림 패턴을 눈과 손에 익힐 수 있습니다.

### 주요 기능
* **실전형 빈칸 채우기 퀴즈**: 자바 스트림의 핵심 메서드를 빈칸으로 두어, 사용자가 직접 올바른 메서드를 입력하도록 합니다
* **즉각적인 코드 검증**: 사용자가 입력한 코드를 실제 자바 문법과 대조하여 정답 여부를 실시간으로 판별합니다.
* **다양한 예제 시나리오**: 리스트 필터링, 객체 변환, 통계 계산 등 자주 쓰이는 다양한 스트림 활용 사례를 문제로 제공합니다.
* **정답 보기 기능**: 막히는 문제가 있을 때 정답을 확인하여 학습 효율을 높일 수 있습니다.
